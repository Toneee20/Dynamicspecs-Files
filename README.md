## How To Deploy on Vercel (Free)

### Step 1 — Upload all 4 files to GitHub
1. Go to your GitHub repository
2. Tap **Add file** → **Upload files**
3. Select all 4 files at once
4. Commit message: `Launch Dynamicspecs website`
5. Tap **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to **vercel.com**
2. Tap **Add New Project**
3. Select this repository → **Import**
4. Leave all settings as default
5. Tap **Deploy**
6. Your site is live in 30 seconds ✅

### Step 3 — Your Live URL
```
https://YOUR-REPO-NAME.vercel.app
```

---

## How To Update The Website

1. Download the new `index.html` file
2. Go to GitHub → tap `index.html` in the file list
3. Tap the **pencil icon** (Edit)
4. Tap the **three dots** → **Delete file** → Commit
5. Upload the new `index.html` → Commit
6. Vercel auto-redeploys in 30 seconds ✅

## Downgrade Vercel to Free Plan

**Important — do this within 14 days to avoid charges:**
1. Go to `vercel.com/account/billing`
2. Tap **Change Plan**
3. Select **Hobby — Free**
4. Confirm ✅

The Hobby plan keeps your site live forever at no cost.
